# Flask CI/CD with Docker, GitLab, and Terraform

This project demonstrates a complete DevOps workflow using Flask, Docker, GitLab CI/CD, and Terraform.

## Features
- Flask web application
- Dockerized deployment
- GitLab CI/CD pipeline
- Infrastructure as Code (Terraform)

## Tech Stack
- Python (Flask)
- Docker
- GitLab CI/CD
- Terraform
- AWS
